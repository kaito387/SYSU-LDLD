设计思路：

设计图片 (assets/design1.jpg)(assets/design2.jpg) 上面画了我对电路的设计，大概想法就是先画出 Q0 Q1 Q2 的 n 和 n+1 时刻之间的关系。根据这个关系，画出卡诺图。画出卡诺图后，得到 Q_0^{n+1} Q_1^{n+1} Q_2^{n+1} 的状态方程：
Q_0^{n+1} = Q_0^n'Q_2^n'
Q_1^{n+1} = Q_0^n\oplus Q_1^n
Q_2^{n+1} = Q_0^nQ_1^n

解出 J-K 驱动方程
J_0 = Q_0^n' Q_2^n'
K_0 = Q_0^n + Q_2^n

J_1 = K_1 = Q_0^n

J_2 = Q_0^n Q_1^n
K_2 = (Q_0^n Q_1^n)' = Q_0^n' + Q_1^n'
然后验证是否可以自启动。经检验 101, 110, 111 这些非法态，根据上面的状态方程，在下一步会分别进入 010(2), 010(2), 100(4) 这三个合法态，可以自启动。

然后 (assets/proteus-simulatioin.jpg) 是我在 proteus 仿真软件上搭建的电路，并且有 logic analyzer 的 debug 图，显示符合预期。(assets/LA.png) 是在电路箱上搭建电路后，真实世界的 LA 图，显示符合预期。

题目如下：

实验器材

双JK触发器74LS73 × 2、门电路若干。

实验内容

用JK触发器设计一个同步五进制加法计数器，并检查能否自启动。

实验要求

（1）写出详细的设计过程。
（2）用Proteus设计电路完成要求。
（3）检查电路能否自启动，若不能，使用将无效状态的次态设置为有效状态的方式使电路能够自启动（例如，将010的次态设为111）。
（4）用实验箱实现电路。
（5）用逻辑分析仪的数字探头通道D₀-D₄测量以下信号：计数脉冲（D₀）、计数器的状态Q_2Q_1Q_0（D₃-D₁）。
（6）提交实验报告的PDF文件。
